# 🚀 Career Compass

> AIがあなたの就職活動をサポートする就活支援アプリ

Career Compass は、ES添削・面接練習・不採用分析・キャリアスコアを一つにまとめたAI就活支援サービスです。

---

## ✨ 主な機能

### 🏢 企業管理
- 応募企業を登録・管理
- 選考状況を記録
- 応募日を管理

### ✍ AI ES添削
- ESをAIがレビュー
- 改善点をフィードバック
- より伝わる文章へブラッシュアップ

### 🎤 AI面接練習
- 想定質問に回答
- AIが回答内容を採点
- 良かった点・改善点を提示

### ❌ AI不採用分析
- 不採用理由をAIが分析
- 次回への改善ポイントを提案

### 📊 Career Score
- 就活状況をスコア化
- 成長を可視化

### 📈 AIレポート
- 就活状況を総合分析
- AIから次のアクションを提案
